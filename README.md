Responsive Registration Form 

This is a simple, stylish and fully responsive **Registration Form** built using HTML5 and CSS3.  
After submitting the form, users are redirected to a Thank You page!  

* Features

- Modern gradient background

- Clean & Responsive Design

- Focus effects & Input animations

- Redirects to Thank You page on form submit

- Media Queries for mobile compatibility

- Semantic HTML structure (Header, Main, Footer)


*Technologies Used

- HTML5
- CSS3
